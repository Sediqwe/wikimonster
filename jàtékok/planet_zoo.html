Planet Zoo fordítása

A PlanetZoo fordításának menete a következő:

  1. Szükséges programok
  2. Kicsomagolás
  3. Fordítás
  4. Visszacsomagolás

Szükséges programok

Cobra-tools az egyetlen szükséges program.

Viszont a Cobra-tools igényel pár programot a futtatáshoz: legfőképpen a Python 3.1 az amit telepíteni kell, legalábbis a fordítások eléréséhez!

/planetzoo/cobra-tools-master.zip

Vagy a mindenkori legfrissebb:
https://github.com/OpenNaja/cobra-tools

https://github.com/OpenNaja/cobra-tools/archive/master.zip

Telepítéséhez előbb fent kell legyen a Python 3.xxx , ott pedig telepítéskor jelöld be “Add Python to PATH” , máskülönben nehezen lesz indítható 

MS Visula C++ ra is szükséged lehet, de azt általában az Win korábban telepítette már.

A Cobra tools a parancssorba beírva így indítható:

py ovl_tool_gui.py

Vagy pedig készítesz egy start.bat fájlt, és bele írod ezt a sort, utánna simán indítható a start.bat-al

File menü → Open → megkeresed a fordítandó fájlokat (ovl)

Edit → Unpak → Kicsomagolás

Edit → Inject → Visszapakolás

A tool több féle játékot ismer, én jómagam a Planet Zoo-t és a Jurassic World Evolution 2 -t próbáltam.

Az eredmény, kb sok 10.0000 TXT fájl, minden fájl 1 - 1 sora a játéknak.
