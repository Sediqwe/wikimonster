Szükséges programok

Cobra-tools az egyetlen szükséges program.

Viszont a Cobra-tools igényel pár programot a futtatáshoz: legfőképpen a Python 3.1 az amit telepíteni kell, legalábbis a fordítások eléréséhez!

/planetzoo/cobra-tools-master.zip

Vagy a mindenkori legfrissebb:
https://github.com/OpenNaja/cobra-tools

https://github.com/OpenNaja/cobra-tools/archive/master.zip

Telepítéséhez előbb fent kell legyen a Python 3.xxx , ott pedig telepítéskor jelöld be “Add Python to PATH” , máskülönben nehezen lesz indítható 

MS Visula C++ ra is szükséged lehet, de azt általában az Win korábban telepítette már.
