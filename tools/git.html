Bevezetés

A Git egy nyílt forráskódú, elosztott verziókezelő rendszer, amelyet Linus Torvalds hozott létre 2005-ben, elsősorban a Linux kernel fejlesztésére. A GitHub egy webalapú platform, amely a Git használatával tárolja a kódot, lehetővé téve a fejlesztők számára a verziókövetést, együttműködést és kódmegosztást. A GitHub a Git-re épít, de számos további funkcióval bővíti, mint például a hibakövetés, a wiki, és a projektmenedzsment eszközök.

Telepítés

Git Telepítése

Windows:

  1. Látogass el a Git hivatalos oldalára.
  2. Kattints a "Download for Windows" gombra.
  3. Futtasd a letöltött telepítőfájlt és kövesd az utasításokat. Az alapértelmezett beállítások általában megfelelőek.
  4. A telepítés után nyisd meg a Git Bash-t a Start menüből.

Linux:

  1. Nyisd meg a terminált.
  2. Futtasd az alábbi parancsot a disztribúciódtól függően:
    • Debian/Ubuntu alapú rendszereken:

sudo apt-get install git

  • Fedora alapú rendszereken:

sudo dnf install git

  • Arch Linux alapú rendszereken:

sudo pacman -S git

GitHub Fiók Létrehozása

  1. Látogass el a GitHub oldalára.
  2. Kattints a "Sign up" gombra és kövesd az utasításokat a fiók létrehozásához.

Git Használata

Első Lépések

Miután telepítetted a Git-t, konfigurálni kell néhány alapvető beállítást:

git config --global user.name "Your Name"

git config --global user.email "your.email@example.com"

Új Repozitórium Létrehozása

  • Navigálj a projekthez tartozó könyvtárba

cd path/to/your/project

  • Inicializálj egy új Git repozitóriumot:

git init

Fájlok Hozzáadása és Első Commit

  • Adj hozzá fájlokat a staging területhez

git add .

vagy minden file:

git add -A

  • Commiteld a változtatásokat

git commit -m "Initial commit"

 

GitHub Használata

Távoli Repozitórium Létrehozása és Kapcsolása

  1. Hozz létre egy új repozitóriumot a GitHub-on: Lépj be a GitHub fiókodba, kattints a "New" gombra, add meg a repozitórium nevét és hozd létre.
  2. Kapcsold a helyi repozitóriumot a távoli repozitóriumhoz:

git remote add origin https://github.com/yourusername/your-repository.git

 

Változtatások Feltöltése GitHub-ra

  1. Pushold a változtatásokat a GitHub repozitóriumba:

git push -u origin master

 

Git és GitHub Hasznossága

Verziókövetés

A Git lehetővé teszi, hogy nyomon kövesd a projekted összes változtatását, visszamenj korábbi verziókhoz, és szükség esetén visszaállítsd azokat.

Együttműködés

A GitHub lehetővé teszi a fejlesztők számára, hogy együtt dolgozzanak ugyanazon a projekten anélkül, hogy félniük kellene az ütközésektől vagy a kód elvesztésétől. A pull request-ek és a code review-k segítik az együttműködést és a kód minőségének javítását.

Biztonsági Másolat

A GitHub tárolja a projektjeidet a felhőben, így ha a helyi másolat megsérül vagy elveszik, mindig visszaállíthatod a kódodat a GitHub-ról.

Főbb Parancsok

Alap Parancsok

  • git init: Új git repozitórium inicializálása.
  • git clone [url]: Egy távoli repozitórium klónozása.
  • git add [file]: Fájl(ok) hozzáadása a staging területhez.
  • git commit -m "message": A staging terület commitolása egy üzenettel.
  • git status: Az aktuális repozitórium státuszának megtekintése.
  • git log: A commit történet megtekintése.

Haladó Parancsok

  • git branch: Lokális ágak listázása.
  • git checkout [branch]: Átváltás egy másik ágra.
  • git merge [branch]: Egy másik ág összeolvasztása az aktuális ággal.
  • git pull: Frissítések letöltése és beolvasztása a távoli repozitóriumból.
  • git push: Lokális változtatások feltöltése a távoli repozitóriumba.
  • git rebase [branch]: Az aktuális ág alapjának megváltoztatása egy másik ág alapján.

Összegzés

A Git és a GitHub használata alapvető a modern szoftverfejlesztésben, mivel hatékony verziókövetést és együttműködési lehetőségeket biztosítanak. A Git segítségével nyomon követheted a projekted változásait, és szükség esetén visszaállíthatod azokat, míg a GitHub lehetővé teszi a kód megosztását és az együttműködést más fejlesztőkkel. A fent bemutatott parancsok és gyakorlatok segítenek az alapok elsajátításában és a hatékony munkavégzésben ezekkel az eszközökkel.
