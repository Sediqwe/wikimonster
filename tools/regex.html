Regex (Reguláris Kifejezések) Részletes Leírás

A reguláris kifejezések (regex) egy erőteljes eszközkészlet a szövegkereséshez és -manipuláláshoz. Ezek a kifejezések lehetővé teszik minták definiálását, amelyeket a szövegben való keresés során lehet használni. A Python nyelvben a re modul biztosítja a reguláris kifejezések használatának lehetőségét.

Alapfogalmak

  1. Minták: A regex minták karakterekből állnak, amelyek meghatározzák, hogy mit keresünk.
  2. Meta-karakterek: Olyan karakterek, amelyek speciális jelentéssel bírnak:
    • .: Bármely karakter
    • ^: Sor eleje
    • $: Sor vége
    • *: 0 vagy több ismétlés
    • +: 1 vagy több ismétlés
    • ?: 0 vagy 1 ismétlés
    • {n}: Pontosan n ismétlés
    • {n,}: Legalább n ismétlés
    • {n,m}: Legalább n, legfeljebb m ismétlés
    • []: Karaktercsoport
    • |: Választás (logikai "vagy")
    • \: Escape karakter

Python Regex Példák

1. Alapvető Használat

python

Code kopieren

import re text = "Hello, my email is example@test.com" pattern = r'\w+@\w+\.\w+'  # Regex minta az email címekhez match = re.search(pattern, text) if match:    print("Talált email:", match.group())

Magyarázat:

  • \w+: Alfanumerikus karakterek (betűk, számok, _), legalább egy alkalommal.
  • @: Az @ karakter.
  • \.: A pont karakter (escape-elve).
  • \w+: Ismétlődő alfanumerikus karakterek a domain névhez.
2. Minta Ellenőrzése

python

Code kopieren

pattern = r'^\d{3}-\d{2}-\d{4}$'  # SSN formátum: XXX-XX-XXXX ssn = "123-45-6789" if re.match(pattern, ssn):    print("Érvényes SSN") else:    print("Érvénytelen SSN")

Magyarázat:

  • ^: A sor eleje.
  • \d{3}: Három számjegy.
  • -: A kötőjel.
  • \d{2}: Két számjegy.
  • -: A második kötőjel.
  • \d{4}: Négy számjegy.
  • $: A sor vége.
3. Több Egyezés Keresése

python

Code kopieren

text = "apple, banana, apple pie, pineapple" pattern = r'apple' matches = re.findall(pattern, text) print("Találtak:", matches)

Magyarázat:

  • re.findall() visszaadja az összes találatot egy listában.
4. Keresés és Csere

python

Code kopieren

text = "Hello World" new_text = re.sub(r'World', 'Python', text) print(new_text)  # Hello Python

Magyarázat:

  • re.sub() egy mintát keres a szövegben, és kicseréli egy új karakterláncra.
5. Karaktercsoportok Használata

python

Code kopieren

text = "A1 B2 C3 D4" pattern = r'[A-Z]\d' matches = re.findall(pattern, text) print("Talált karakterek:", matches)  # ['A1', 'B2', 'C3', 'D4']

Magyarázat:

  • [A-Z]: Bármely nagybetű.
  • \d: Bármely számjegy.
6. Választás és Csoportosítás

python

Code kopieren

text = "cat, bat, rat" pattern = r'[cbr]at' matches = re.findall(pattern, text) print("Talált állatok:", matches)  # ['cat', 'bat', 'rat']

Magyarázat:

  • [cbr]: Választható karakterek (c, b vagy r), majd "at".
7. Escape Karakterek

python

Code kopieren

text = "3.14 is a number." pattern = r'\d+\.\d+' match = re.search(pattern, text) if match:    print("Talált szám:", match.group())  # 3.14

Magyarázat:

  • \d+: Számjegyek.
  • \.: A pontot escape-eljük, hogy mintaként kezeljük.

Összegzés

A reguláris kifejezések (regex) rendkívül hasznosak a szövegfeldolgozás során. A Python re moduljának segítségével könnyen kereshetünk mintákra, ellenőrizhetjük a formátumokat, helyettesíthetünk karakterláncokat és összetett kereséseket végezhetünk. A fenti példák bemutatják a regex alapjait, de a minták testreszabása és finomítása számos lehetőséget kínál a fejlesztők számára.
